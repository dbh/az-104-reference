# azure app svc - custom domain names

* basic / default 
  * [app].azurewebsites.net
* custom domains 
  * requires paid tier
  * reserve domain name
  * create DNS records domain to your azure web app
  * A record for IP
  * CNAME for domain to another domain
  * enable custom domain
